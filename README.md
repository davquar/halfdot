HalfDot is a minimalistic **FOSS Android client for [Umami](https://github.com/umami-software/umami)**: an open-source and privacy-focused web analytics platform.

HalfDot is:

* Free.
* Open source.
* Without tracking.
* Without ads.
* Material Design 3.
